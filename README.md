# Python Portfolio — Andre Weissmann

Python projects built around real healthcare and operations data.

Focus areas: exploratory data analysis, data cleaning automation, BMI and biometric calculations, and workflow tooling inspired by real healthcare operations work.

## What's here

- **BMI Calculator** — patient biometric analysis tool. Goes beyond a basic calculator: preset patient profiles, metric/imperial toggle, BMI category classification, and a clean output view built for readability.
- Automation scripts for operations and data prep workflows

## Tools

Python · pandas · NumPy · Matplotlib · Jupyter Notebooks
